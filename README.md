# Arun's Homepage

### Run Development Environment on Your Local Machine

```bash
$ git clone https://github.com/varararun/varararun.github.io.git
$ cd varararun.github.io
$ nvm use
$ npm start
```  

Prerequisites

- `git` for source code management
- `nvm` for node version management
